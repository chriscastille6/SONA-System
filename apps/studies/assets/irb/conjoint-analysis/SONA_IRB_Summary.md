## Protocol Summary
1. **Participants**
   - Undergraduate students enrolled in Dr. Meder's Labor Economics course at Nicholls State University
   - All participants are 18+ and participation is voluntary

2. **Procedure**
   - Consent and demographic intake via static HTML/CSS/JavaScript survey (`student_survey.html`)
   - Eight choice tasks comparing job packages built from the latest baseline values
   - Retention probability questions for baseline, preferred, training-focused, and manager-focused packages
   - Group work to design a class-wide package under a $59K budget constraint
   - Final choice round followed by comprehensive assessment and class discussion of aggregate results

3. **Data Collection Notes**
   - Tool: Static HTML/CSS/JavaScript application (September 2025 build)
   - Platform: Local server or standard web hosting
   - Storage: Anonymous—data remain in-browser unless exported by instructor
   - Screenshots included in IRB to document user flow; `03_choice_task_options_A_and_B.png` refreshed Oct 30, 2025 to reflect current baseline values

## Baseline Package & Attribute Updates (Oct 30, 2025)

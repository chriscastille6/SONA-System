03 · Architecture

Where packages and plots may run

Control Plane · Cloud

May receive

  • Empty schemas (column names / dtypes)
  • “Draft a seaborn heatmap of gpa vs ets_score”
  • Returned code snippets only

Must not receive

  • CSV / frame rows / chart image pixels of real data
  • Disney/real names, emails, student IDs
  • Joint demo×outcome schemas (pre–break-glass)
Data Plane · Campus

Runs locally

  • pandas, numpy, scipy — analysis
  • matplotlib, seaborn — figures in this deck
  • Human-in-the-Loop gate before execution
  • Figures retained under university custody
PoC script: scripts/decoupled_ai_protocol_poc.py
In plain English

Two rooms. Cloud room: AI sees only an empty spreadsheet outline. Campus room: our tools make the charts. Student files stay in the campus room.

09 · Special written OK

When background + grades charts are allowed

Required log fields

  • timestamp_utc
  • approver (name/role)
  • reason
  • analysis_requested
  • tables_linked
  • authorization_id

Still forbidden after approval

  • Uploading linked rows to Gemini/OpenAI
  • Including PII vault columns in extracts
  • Skipping Human-in-the-Loop attestation
  • Treating synthpop tables as “anonymous enough” to send off-campus
Break-glass unlocks local linkage. It does not authorize cloud ingestion.
In plain English

A special written OK records who approved, why, and when. It unlocks joining files on campus. It does not allow sending student data to the cloud.
